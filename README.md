# webapp-template
Template for Web application. This template uses Ractive.js + Backbone.js(Router) and written in TypeScript. 

# How to compile .ts files

    $ cd js
    $ make app
    or
    $ tsc app.ts -out app.js

`make` command just executes `tsc app.ts -out app.js`
